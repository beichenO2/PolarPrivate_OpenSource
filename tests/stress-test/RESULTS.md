## 一、视觉模型（PDF/PPT 图片输入）

### 测试数据
- 来源：`~/Desktop/3_2nd/雷达实验/` (2 PDF + 3 PPT)
- PDF 页面转 PNG 图片，共 44 张（150dpi，总计 21.6MB）
- 内容：雷达实验结果图表（测距/测速/CFAR/测角）

### 结果

| 模型 | 1页 | 3页 | 5页 | 10页 | 20页 | 44页(全部) |
|------|-----|-----|-----|------|------|-----------|
| **xopkimik26** (Kimi-K2.6) | 10.8s OK | 31s OK | 500 Error | 500 Error | - | - |
| **qwen3.7-plus** | 37.9s OK | 52.9s OK | 55.7s OK | 73.6s OK | 90.4s OK | 104.1s OK |
| **qwen3-vl-flash** | 3.9s OK | 24.6s OK | 31.8s OK | 27.3s OK | 22.4s OK | 30.3s OK |

### 精度评估
- **文档计数**：3 模型均正确识别文档数量
- **标题识别**：准确读取每页标题（如 "1/21 报告2 测距 · data_50.bin"）
- **参数提取**：能准确提取带宽(B)、距离(R)、信噪比(dB)等参数
- **图表理解**：能区分距离维功率谱、速度维功率谱、PPI 图等
- **Kimi-K2.6 限制**：xfyun 代理限制最多 3-4 张图片输入

### 结论
- **最佳视觉模型**：qwen3-vl-flash（速度最快，44页仅30s）
- **最强综合**：qwen3.7-plus（44页 104s，描述最详细）
- **Kimi-K2.6**：文本理解极强，但通过 xfyun 代理的多图支持有限
